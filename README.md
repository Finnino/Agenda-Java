# ☕ Agenda Java

Sistema de gerenciamento de contatos desenvolvido durante a graduação em Análise e Desenvolvimento de Sistemas.

## 🚀 Tecnologias

- Java
- Programação Orientada a Objetos
- Collections
- Git

## 📚 Funcionalidades

- Cadastro de contatos
- Atualização de contatos
- Exclusão de contatos
- Organização das informações

## 🎯 Objetivo

Aplicar os conceitos estudados durante a graduação em um projeto prático, reforçando conhecimentos em Java e desenvolvimento de software.
